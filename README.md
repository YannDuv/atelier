# 🛠 Atelier

Atelier is a web component library that use simplicity for fast building applications.

## 🎁 Usage

```bash
npm install @yannduval/atelier
```

```typescript
import "@yannduval/atelier/dist/bundle";
```

## 👨‍💻Development

### 🏠Installation

```bash
npm install
```

### 🏃Run

```bash
npm start
```
